# 遥测对标分析模板

## 标准输出结构（三层递进）

### 第一层：全局速度结构（宏观定调）
**目标**：一句话说清"谁快、快多少、快在哪里"

**数据来源**：
- `sectors`: S1/S2/S3分段差
- `straights`: 最高速、油门开度
- `lap_time`: 总圈时差距

**输出要求**：
- 用赛段时间差定位优势分布（"S1快0.3s + S3快0.1s，S2持平"）
- 结合赛道特性解读（如"S1是高速弯区→下压力优势"）
- 指出优势来源的赛道物理特征，不是简单报数字

**示例**：
> 乔治·拉塞尔整圈快0.4秒，优势集中在S1(+0.3s)和S3(+0.1s)。铃鹿S1是连续高速弯区，这意味着下压力设定更激进；S3包含130R和chicane，PU部署效率更高。

---

### 第二层：分段优势来源（中观归因）
**目标**：将赛段差异拆解为具体的弯角/直线/技术动作差异

**数据来源**：
- `corners`: 刹车点、最低速、出弯速差异
- `straights`: 最高速差、油门开度差
- 赛道知识库的 `sectors.key_corners` 和 `key_analysis_factors`

**输出要求**：
- 按赛道物理分区分析（"高速弯→下压力"、"重刹区→刹车效率"、"直线→PU/阻力"）
- 每个分区至少引用1个具体数据点
- 区分"入弯优势"（刹车点/最低速）和"出弯优势"（出弯速/油门开度）

**示例**：
> S1的优势来自T1-T6 Esses段：拉塞尔在T3(Degner)的最低速高4km/h，出弯速高6km/h，说明弯中下压力更足。但T1刹车点早8m——不是刹车能力差，而是入弯速度更高需要更早减速。S3的130R全油门区间，拉塞尔油门开度98% vs 安东内利95%，最高速差3km/h，指向PU部署策略差异。

---

### 第三层：因果归因链（微观论证）
**目标**：建立"数据→物理原因→设定/策略推断"的完整逻辑链

**数据来源**：
- `tyre`: 圈时标准差、衰退斜率
- `stint_degradation`: 分段衰退
- `key_laps`: 关键时刻
- 赛道知识库的 `tire_severity`、`fuel_correction_s_per_kg`、`typical_pit_loss_s`

**归因逻辑链标准格式**：
```
观测数据 → 物理现象 → 可能原因(按概率排序) → 验证方法
```

**示例**：
> **观测**：安东内利S2圈时标准差0.45s，拉塞尔仅0.25s。
> **物理现象**：S2圈速波动大→弯角速度不稳定。
> **可能原因**：
> 1. (高概率) S2 Spoon弯出弯牵引力不稳定→机械抓地力设定偏激进
> 2. (中概率) 轮胎温度窗口窄→弯中温度波动导致抓地力变化
> 3. (低概率) 车手风格差异→安东内利入弯更激进导致出弯修正
> **验证**：检查stint_degradation中S2对应圈号的退化斜率是否高于S1/S3。

## 因果归因的标准逻辑链

### 速度差异归因决策树

```
A比B快 → 哪个赛段快？
├── 高速弯快 → 检查min_speed_delta
│   ├── A最低速高 → 下压力优势（弯中速度高）
│   │   └── 验证：直线最高速是否较低？(阻力代价)
│   └── A出弯速高 → 牵引力/机械抓地力优势
│       └── 验证：油门开度是否更早达到100%？
├── 重刹区快 → 检查brake_point_delta
│   ├── A刹车点晚 → 刹车效率/信心优势
│   │   └── 验证：进站前的圈是否刹车点一致？(排除轮胎影响)
│   └── A最低速低但出弯快 → 更激进的弯心路线
│       └── 验证：出弯油门开度是否更高？
├── 直线快 → 检查top_speed_delta
│   ├── A最高速高 → 低阻力设定或PU部署优势
│   │   └── 验证：高速弯是否相应慢？(阻力-下压力权衡)
│   └── A油门开度高 → 更有效的牵引力控制
│       └── 验证：出弯速是否也高？(连贯性)
└── 轮胎管理好 → 检查degradation_slope
    ├── A衰退斜率低 → 驾驶保护/设定保护
    │   └── 验证：stint初圈是否相应慢？(性能换耐久)
    └── A标准差低 → 驾驶一致性更好
        └── 验证：排除交通和DRS状态的影响
```

### 归因注意事项

1. **单一数据不可定论**：每个结论至少需要2个独立数据点支撑
2. **排除替代假说**：提出归因后必须说明排除了什么其他可能
3. **标明置信度**：用"高/中/低"标注推断的可信程度
4. **区分设定vs风格**：同一现象可能是赛车设定差异，也可能是驾驶风格差异，需给出判断依据
5. **避免后此谬误**：A事件在B之后发生，不代表A导致B（如：进站后圈速快不一定是新胎优势，可能是燃油负载降低）

## 5个专业级输出范例

### 范例1：铃鹿高速弯下压力对决
> 乔治·拉塞尔 vs 安德雷亚·基米·安东内利 | 铃鹿排位赛 | 差距0.3s
>
> **一句话结论**：拉塞尔赢在下压力设定——S1 Esses的弯中速度优势(最低速平均+4km/h)贡献了0.3s中的0.25s，但代价是S3直线速度低2km/h。
>
> **速度结构**：S1(+0.25s)→S2(-0.02s)→S3(+0.07s)。铃鹿S1是连续高速弯，下压力决定速度；S3是130R+chicane，需PU部署效率。
>
> **分段拆解**：T3 Degner弯，拉塞尔最低速148km/h vs 安东内利144km/h，出弯速差6km/h。这不是刹车差异——两人刹车点几乎相同——而是弯中下压力更足导致速度维持能力更强。T7 Hairpin反而安东内利快0.02s，说明机械抓地力（低速弯）方面梅赛德斯W207基础水平接近，差异在高速弯的空力效率。
>
> **因果链**：最低速+4km/h(数据) → 弯中抓地力更足(物理) → 更高的下压力设定(推断,高置信) → 验证：直线最高速低2km/h(阻力代价吻合)。安东内利的驾驶风格更依赖弯角速度平衡，而拉塞尔选择了铃鹿传统的高下压力路线。

### 范例2：巴林轮胎退化管理决胜
> 马克斯·维斯塔潘 vs 兰多·诺里斯 | 巴林正赛 | 最终差距8.2s
>
> **一句话结论**：维斯塔潘的硬胎stint衰退斜率仅0.03s/圈，诺里斯0.06s/圈——同样的硬胎，不同的命运，驾驶保护差异决定了8秒的胜差。
>
> **关键时刻**：第28圈，诺里斯进站换中性胎，此时他的软胎衰退斜率已达0.12s/圈。维斯塔潘多跑3圈(L31进站)，利用软胎最后性能建立1.5s缓冲后安全进站。
>
> **轮胎因果链**：诺里斯硬胎退化0.06s/圈(数据) → 后轮过热(物理) → 出弯牵引力过剩导致后轮滑移(推断,高置信) → 验证：诺里斯S3(低速弯+出弯加速)圈时波动比S1高30%。维斯塔潘则牺牲了S1的高速弯速度(入弯更保守)，换取了后轮保护。

### 范例3：蒙扎直线速度对决
> 夏尔·勒克莱尔 vs 奥斯卡·皮亚斯特里 | 蒙扎正赛 | 差距1.8s
>
> **一句话结论**：蒙扎是直线速度的终极考验——勒克莱尔最高速342km/h vs 皮亚斯特里335km/h，7km/h的差距在7圈DRS直道累计成1.8秒。
>
> **速度结构**：S1(-0.05s)→S2(+0.02s)→S3(+0.11s)。S1包含Prima Variante重刹，皮亚斯特里刹车更晚(晚4m)；但S3的长直道，勒克莱尔的低阻力设定带来0.11s优势。
>
> **归因链**：最高速+7km/h(数据) → 阻力水平更低(物理) → 极低下压力设定(推断,高置信) → 验证：S1 Lesmo弯中速度低3km/h(下压力代价吻合)。法拉利选择了蒙扎传统的"最小下压力"路线，而迈凯伦在弯角速度和直线速度之间选择了更保守的平衡点。

### 范例4：摩纳哥交通与overcut
> 刘易斯·汉密尔顿 vs 乔治·拉塞尔 | 摩纳哥正赛 | 差距3.1s
>
> **一句话结论**：摩纳哥的策略只有一个原则——赛道位置就是一切。汉密尔顿L18 overcut拉塞尔2.1秒，靠的是对手新硬胎warm-up期的2圈宝贵时间。
>
> **关键时刻**：L18拉塞尔进站换硬胎(pit loss 18s)，出站后L19-L20硬胎warm-up期间每圈慢0.8s。汉密尔顿L20进站，利用这两圈的赛道位置优势+旧软胎最后冲刺，overcut净赚1.2s。
>
> **归因链**：overcut成功(结果) → 硬胎warm-up慢2圈(数据) → 摩纳哥低温+低表面粗糙度=硬胎难进窗口(物理) → 低退化赛道+硬胎warm-up差=overcut条件完美(推断,高置信)。

### 范例5：斯帕雨战策略
> 兰多·诺里斯 vs 马克斯·维斯塔潘 | 斯帕正赛(雨) | 差距4.7s
>
> **一句话结论**：斯帕的天气是最大的策略变量——诺里斯在L12换半雨胎，比维斯塔潘早2圈，抓住了赛道从湿到半湿的转换窗口，每圈快1.2s的2圈优势累计成2.4s基础。
>
> **关键时刻**：L10赛道开始变干，诺里斯L12进站换半雨胎。维斯塔潘L14才换，这两圈里诺里斯半雨胎在逐渐变干的赛道上每圈快1.2s。这不是"更好的赛车"——是"更好的天气判断"。
>
> **归因链**：L12换胎vs L14(决策差异) → 赛道转换窗口早2圈抓住(策略判断) → 半雨胎在微湿条件下性能窗口更宽(物理特性) → 斯帕不同区段干湿不同(赛道特性)，趁Pouhon段变干提前换胎是正确的(推断,高置信)。
