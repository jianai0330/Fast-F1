# F1 遥测对标分析技能 (Telemetry Benchmark Analysis)

## 元信息

- **技能ID**: f1-telemetry-analysis
- **版本**: 1.0.0
- **适用场景**: 排位赛/正赛单圈对标、长距离节奏对比、遥测数据解读
- **前置依赖**: FastF1遥测数据（Speed, Throttle, Brake, nGear, DRS, Distance）
- **核心方法论**: 速度曲线对标 → 参数分解 → 物理机制解读 → 竞争力结论

---

## 1. 分析流程总览

### 1.1 标准分析管线

```
遥测数据加载
    │
    ▼
距离对齐（以距离为X轴叠加）
    │
    ▼
速度差曲线绘制（ΔSpeed vs Distance）
    │
    ▼
定位时间差异的物理位置（弯角/直线段映射）
    │
    ▼
四阶段分解：制动 → 弯心 → 出弯 → 直线
    │
    ▼
物理机制归因（空力设置 / 驾驶风格 / 轮胎 / 能量部署）
    │
    ▼
竞争力结论（量化 + 因果链 + 置信度）
```

### 1.2 数据准备清单

在开始分析前，必须确认以下数据可用且质量合格：

| 数据项 | 来源 | 质量检查 |
|--------|------|----------|
| 车速遥测 | `car_data` / `get_car_data()` | 距离采样率≥10Hz，无大段NaN |
| 油门/刹车 | `get_telemetry()` | 刹车信号0/1无中间值漂移 |
| DRS状态 | `get_telemetry()` | 确认DRS开合与赛道位置一致 |
| 档位 | `get_telemetry()` | 换挡点应在直线段，弯中换挡可能异常 |
| 弯角信息 | `circuit_info.corners` | **注意：2026赛季此数据可能全为NaN，需降级处理** |
| 赛道距离 | `session.laps` | 确认总圈距离与赛道图一致 |

### 1.3 数据清洗规则

1. **排除进站圈**：`LapStatus != PitIn & LapStatus != PitOut`
2. **排除安全车圈**：`TrackStatus != '4' & TrackStatus != '6'`
3. **排除异常快圈**：圈时 < 中位数 * 0.97 的圈可能是数据错误
4. **排除异常慢圈**：圈时 > 中位数 * 1.10 的圈含进站/安全车/慢车阻挡
5. **燃油负载校正**：如可能，按每圈约-0.03s/圈校正直线速度

## 2. 关键指标解读规则

### 2.1 制动点差异

**物理机制**：制动点位置反映车手对前轮抓地力的信心和底盘平衡设置。

| 观测 | 物理解读 | 可能原因 |
|------|----------|----------|
| 制动点提前（比对手晚5m+进入制动区） | 保守/信心不足 | 前轮锁定风险高、轮胎温度不足、制动衰退 |
| 制动点延后（比对手晚5m+进入制动区） | 激进/对前叉刚度信任 | 高下压力提供额外制动力、车手风格、制动效率高 |
| 制动强度曲线陡峭 | 硬制动设定 | 制动力分配偏前、ABS-like调制减少 |
| 制动强度曲线平缓 | 渐进制动 | 轮胎保护策略、后轮锁定风险规避 |

**分析要点**：
- 制动点差异需结合弯角类型解读：低速弯（<100km/h）制动差异意义小，高速弯（>200km/h）制动差异意义大
- 制动点延后≠更快：如果弯心速度因此降低，净效果可能为负
- 每提前1m制动点，在120km/h弯角约损失0.03s

### 2.2 弯心速度差异

**物理机制**：弯心速度反映车辆在最大横向加速度下的平衡状态。

| 观测 | 物理解读 | 可能原因 |
|------|----------|----------|
| 弯心速度高3+km/h | 下压力设定更激进 | 高翼展/更大攻角→更多弯中下压力 |
| 弯心速度低但出弯快 | 循迹制动+早开油门 | V型过弯vs U型过弯的驾驶风格差异 |
| 弯心速度差异随圈数增大 | 轮胎退化不一致 | 一车轮胎在温度窗口内，另一车已过热 |
| 高速弯弯心速度差>5km/h | 空力效率差异显著 | 底板扩散器效率或悬挂几何差异 |

**分析要点**：
- 弯心速度必须结合弯角类型分析：
  - 低速弯（<100km/h）：机械抓地力主导，下压力影响<10%
  - 中速弯（100-200km/h）：下压力与机械抓地力各约50%
  - 高速弯（>200km/h）：下压力主导，占比>70%
- 弯心速度差1km/h在不同弯角的时间影响：
  - 低速弯：约0.01-0.02s
  - 中速弯：约0.02-0.04s
  - 高速弯：约0.03-0.06s

### 2.3 出弯加速差异

**物理机制**：出弯加速反映后轮牵引力、油门信心和能量部署策略。

| 观测 | 物理解读 | 可能原因 |
|------|----------|----------|
| 早开油门（弯心前已全油门） | 对后轮牵引力有信心 | 牵引力控制系统调校、后轮温度窗口、差速器锁定率 |
| 晚开油门（弯心后仍部分油门） | 后轮打滑风险规避 | 后轮过热、扭矩过剩、差速器设置偏开放 |
| 出弯加速曲线陡峭 | MGU-K部署积极 | 2026规则下50/50功率比，电能释放策略激进 |
| 出弯加速曲线平缓 | MGU-K节省模式 | 保存电能用于直线段/防守、电池SoC不足 |

**分析要点**：
- 出弯加速与弯心速度强耦合：弯心快→出弯角度好→加速快→连锁效应
- 出弯0-200m的加速差异往往比最高速差异更有意义（影响下一弯角的入弯速度）
- 2026新规：MGU-K功率提升至200kW+，出弯电能部署成为关键变量

### 2.4 最高速差异

**物理机制**：最高速反映阻力-下压力权衡和动力单元输出。

| 观测 | 物理解读 | 可能原因 |
|------|----------|----------|
| 最高速高3+km/h且弯中慢 | 低阻力设定 | 减少翼展→直线快但弯中少下压力 |
| 最高速低但弯中快 | 高下压力设定 | 增加翼展→弯中快但直线阻力大 |
| 最高速相近但到达时间不同 | 加速曲线差异 | MGU-K部署时机不同或牵引力差异 |
| DRS开启时速度差特别大 | DRS效率差异 | 翼展设计差异、DRS开度效果不同 |

**分析要点**：
- 最高速必须标注DRS状态：DRS开vs关的速度差可达10-15km/h
- 速度陷阱（Speed Trap）位置影响解读：终点线前vs测量点位置不同
- 尾流效应：跟车时可获得额外3-5km/h速度提升

## 3. 速度曲线对标方法论

### 3.1 距离对齐

```
核心原则：以距离（Distance）为X轴，而非时间

原因：
1. 两车在赛道同一点的物理条件相同（弯角、坡度、路面）
2. 时间轴对齐无法反映"谁在哪里快"
3. 距离轴可直接映射到赛道图上的具体位置

实现：
- 获取两车的遥测数据
- 使用 Distance 列作为X轴
- Speed 列作为Y轴
- 在同一图上叠加绘制
- 绘制 ΔSpeed = Speed_A - Speed_B 在下方子图
```

### 3.2 时间差异的物理位置定位

```
步骤：
1. 计算累计时间差：ΔT(D) = ∫[0,D] (1/Speed_B - 1/Speed_A) dD
2. 定位ΔT变化最陡的位置 → 最大时间差异来源
3. 将这些位置映射到弯角编号（如T1-T3）
4. 分段汇总：每个弯角+直线段贡献了多少时间差
```

### 3.3 四阶段分解法

对每个弯角，将过弯过程分解为四个阶段：

| 阶段 | 定义 | 关键参数 | 时间贡献估算 |
|------|------|----------|-------------|
| 制动 | 从全油门到最大制动压力 | 制动点距离、制动减速度 | 差异占比10-20% |
| 入弯/弯心 | 从释放制动到弯心 | 弯心速度、横向加速度 | 差异占比30-50% |
| 出弯 | 从弯心到全油门 | 油门应用点、牵引力 | 差异占比20-35% |
| 直线 | 全油门到下一弯制动点 | 最高速、DRS、MGU-K | 差异占比15-30% |

**注意**：百分比范围为典型值，不同赛道差异很大。低速赛道弯角贡献占比更高，高速赛道直线贡献占比更高。

## 4. 正确 vs 错误示例

### 4.1 速度差描述

❌ **错误**："A在S1快了0.2秒"
- 问题：纯数字罗列，无物理机制，无位置信息

✅ **正确**："A在S1的T1-T3高速弯组合中入弯速度平均高3km/h，说明下压力设定更激进，净收益0.14s；直线段DRS贡献额外0.06s"
- 优点：具体位置 + 物理机制 + 量化分解

### 4.2 制动点分析

❌ **错误**："B制动更早，说明他更保守"
- 问题：过度简化，制动早可能是策略选择而非保守

✅ **正确**："B在T1制动点提前约8m（约0.04s损失），但弯心速度仅低1km/h，表明采用循迹制动策略——延长制动区以维持更高的弯中横向加速度，这在他第12-18圈的长距离stint中显示出更优的轮胎管理"
- 优点：量化 + 策略意图解读 + 长距离验证

### 4.3 最高速对比

❌ **错误**："A最高速比B快5km/h，说明动力更强"
- 问题：忽略了阻力设定差异，可能A只是低阻力配置

✅ **正确**："A在主直线最高速高出5km/h（328 vs 323km/h），但T4-T6弯中速度低4km/h。综合判断为低阻力设定：减翼展换直线速度，在Spa/Monza有优势，但Suzuka/Monaco将付出代价。对比两车S3末段加速曲线斜率一致，排除了PU输出差异的解释"
- 优点：排除替代假说 + 赛道适用性 + 具体数据

### 4.4 DRS影响

❌ **错误**："A在正赛中使用DRS获得了很多时间"
- 问题：未量化，所有车手都有DRS

✅ **正确**："A在DRS激活区（T1直线后段）获得0.08s相对优势，但这是因为A的出弯速度比B高2km/h，使得DRS开启时间提前约15m。DRS本身是均等工具，差异来源于DRS前的速度差"
- 优点：区分了DRS本身和DRS前的速度差

## 5. 常见陷阱与Gotchas

### 5.1 数据质量陷阱

| 陷阱 | 说明 | 应对策略 |
|------|------|----------|
| 进站圈数据混入 | 进站圈圈时暴增20-25s，遥测数据不连续 | 严格过滤 `PitIn/PitOut` 状态圈 |
| 安全车圈 | VSC/SC下所有车减速，遥测对比无意义 | 过滤 `TrackStatus=4/6` 的圈 |
| 燃油负载差异 | 排位赛Q1/Q2/Q3燃油量不同，正赛每圈减轻约1.5kg | 排位赛按Q3对比；正赛做每圈校正（约0.03s/圈） |
| DRS在VSC期间 | VSC下DRS不可用，速度差可能误导 | 检查TrackStatus与DRS状态的交叉 |
| 遥测采样不同步 | 两车遥测距离点不完全对齐 | 插值对齐后再比较 |

### 5.2 分析逻辑陷阱

| 陷阱 | 说明 | 应对策略 |
|------|------|----------|
| 相关性≠因果 | 制动点早和圈时慢同时出现，但不一定因果 | 需要物理机制解释，不能仅靠统计 |
| 幸存者偏差 | 只分析完赛车的数据，退赛车可能有不同策略 | 标注分析范围，提及其他车手的策略差异 |
| 单圈过度推断 | 一圈的数据不能代表整体表现 | 至少3圈一致性验证 |
| 忽略赛道进化 | 赛道橡胶铺设后圈时自然改善 | 对比同圈数位置的车手，而非绝对圈时 |
| 出站圈轮胎温度 | 出站圈前3-4圈轮胎未达工作温度 | 新胎stint前3圈不做退化分析 |

### 5.3 2026规则特殊性

**50/50 ICE-MGU-K功率比变化**：
- 2026年MGU-K功率从120kW提升至200kW+，ICE功率相应降低
- **新分析维度**：能量部署一致性
  - 两车在直线段的加速曲线可能因MGU-K释放策略不同而差异显著
  - 需要关注：哪辆车在出弯时MGU-K全释放？哪辆车在直线中段释放？
  - 消耗型释放（出弯全开）vs 均匀释放的圈时差异可达0.1-0.2s

**能量回收限制**：
- 制动回收能量上限变化影响弯角间的能量储备
- 连续弯角组（如Suzuka S弯）后的直线段，能量可能已耗尽
- 需要在分析中标注"能量状态"：满电/半电/空电

**主动空力规则**：
- 2026赛季可能的主动空力元素增加了分析复杂度
- 不同模式下空力效率差异可能误归因为"设定差异"
- 需确认数据中是否包含空力模式信息

### 6.2 量化精度规则

- 圈时差：精确到0.01s
- 速度差：精确到1km/h
- 制动点差：精确到1m
- 弯心速度差：精确到0.5km/h
- 退化率：精确到0.001s/圈
- 所有量化结论必须标注数据来源圈数范围

### 7.2 速度曲线叠加对比

```python
import matplotlib.pyplot as plt

lap_a = session.laps.pick_driver('VER').pick_fastest()
lap_b = session.laps.pick_driver('NOR').pick_fastest()

tel_a = lap_a.get_telemetry()
tel_b = lap_b.get_telemetry()

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 8), height_ratios=[3, 1])

# 速度叠加
ax1.plot(tel_a['Distance'], tel_a['Speed'], label='VER', color='blue')
ax1.plot(tel_b['Distance'], tel_b['Speed'], label='NOR', color='orange')
ax1.set_ylabel('Speed (km/h)')
ax1.legend()

# 速度差
ax2.plot(tel_a['Distance'], tel_a['Speed'] - tel_b['Speed'], color='green')
ax2.axhline(y=0, color='gray', linestyle='--')
ax2.set_ylabel('ΔSpeed (km/h)')
ax2.set_xlabel('Distance (m)')
```

### 7.4 制动点分析

```python
# 定位制动点：从全油门到开始制动的位置
def find_braking_points(telemetry):
    """识别每个制动事件的起点"""
    braking = telemetry[telemetry['Brake'] > 0]
    # 找到制动开始点（从非制动到制动的转换点）
    brake_starts = []
    prev_braking = False
    for idx, row in telemetry.iterrows():
        if row['Brake'] > 0 and not prev_braking:
            brake_starts.append(row['Distance'])
        prev_braking = row['Brake'] > 0
    return brake_starts
```
