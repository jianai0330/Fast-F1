## 🟠 P1 — 体验核心问题（影响能不能用）

### P1-1 正赛圈时图表可读性大改造
**问题：** 20个车手的点和线全挤在一起，完全看不清。
**方案：**
- 默认只显示前 5 名，其余折叠（可点击展开）
- 增加"高亮"交互：点击某条线，其他线变灰
- 异常圈（pit stop、SC、VSC）用特殊图标标注，不影响主线
- Y 轴改为"与最快圈的差值"（+0.0s / +1.2s），更直观
- 加入 pit stop 时机标注竖线
- 图表高度从 500rpx 改为 600rpx，给更多空间

### P1-2 页面整体 UI 品质提升
**问题：** 整体像初学者作品，缺乏层次感和精致度。
**方向：**
- 统一组件规范：卡片圆角、间距、字体层级
- 加载状态改为骨架屏（Skeleton），不要"加载中..."文字
- 页面切换加入过渡动画（navigateTo 的进入动画）
- 列表项加 pressEffect（按下高亮反馈）
- 空状态页面设计（暂无数据时的占位图）
- 顶部 header 区域重设计，加入赛旗/渐变背景

## 🟡 P2 — 功能增强（做完 P0/P1 再做）

### P2-1 赛历页可视化升级
**问题：** 现在只是一个列表，太朴素。
**方案：**
- 每个赛事卡片显示：
  - 国旗 emoji（🇦🇺 🇨🇳 等）
  - 距今倒计时：「还有 X 天」（已结束则显示「已结束」）

- 支持手势缩放（pinch zoom）某段赛道距离

### P2-3 积分榜增强
